# قائمة اختبار قبل الرفع النهائي

## بعد رفع GitHub وVercel

- [ ] افتح `/setup` وأنشئ أول إدارة.
- [ ] سجل دخول الإدارة.
- [ ] سجل مورد جديد برقم هاتف وبريد اختياري.
- [ ] اعتمد المورد من الإدارة.
- [ ] سجل دخول المورد.
- [ ] أضف منطقة توصيل: محافظة / مركز / قسم / رسوم / مدة.
- [ ] أضف منتج بصورة 600×600 وأسعار القطاعي والجملة وجملة الجملة.
- [ ] اعتمد المنتج من الإدارة.
- [ ] افتح السوق كزائر وتأكد من ظهور المنتج.
- [ ] جرّب البحث بالمحافظة والمركز والقسم.
- [ ] سجل عميل جديد.
- [ ] أضف المنتج للسلة.
- [ ] جرّب السلة العادية في منطقة يغطيها المورد.
- [ ] جرّب السلة العادية في منطقة لا يغطيها المورد وتأكد أن الطلب ممنوع.
- [ ] جرّب السلة المميزة وتأكد من إضافة الرسوم.
- [ ] أنشئ طلبًا.
- [ ] تأكد أن الطلب يظهر للعميل والمورد والإدارة.
- [ ] حدّث حالة الطلب والتوصيل.
- [ ] سجّل دفعة عمولة من حساب المورد.
- [ ] اعتمد الدفعة من الإدارة.
- [ ] راجع المتبقي والمدفوع وصافي المورد.
