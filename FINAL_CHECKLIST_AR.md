# قائمة الفحص النهائية

- اللوجو موجود في الهيدر والفوتر وصفحات التسجيل.
- لا توجد عبارات خاصة بإنشاء الملف أو ملاحظات داخلية في الواجهة العامة.
- الصفحة العامة منفصلة عن صفحات العميل والمورد والإدارة.
- تسجيل العميل منفصل عن تسجيل المورد.
- بعد تسجيل الدخول تظهر الصفحات حسب نوع الحساب فقط.
- المنتجات تدعم صورة وسعر قطاعي وسعر جملة وسعر جملة الجملة.
- الإعدادات المالية داخل لوحة الإدارة فقط.
- البناء يعمل على Vercel باستخدام `npm run build` ومجلد `public`.
