# قائمة الفحص النهائية

- اللوجو موجود في الهيدر والصفحات الرئيسية.
- الألوان محفوظة من هوية Tager: أخضر/تركواز مع برتقالي وخلفية فاتحة.
- الصفحة الرئيسية مبسطة وتعرض الصفحات المهمة.
- المنتج يدعم صورة وسعر قطاعي وسعر جملة وسعر جملة الجملة.
- العميل لديه صفحات شراء وسلة وتتبع ودعم.
- المورد لديه صفحات منتجات وتغطية وطلبات وحساب مالي.
- الإدارة لديها صفحات اعتماد وتشغيل ومالية وإعدادات وبيانات.
- Build يعمل بنجاح.
- Output Directory هو `public`.
