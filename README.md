# Tager invoice and purchase order templates

GitHub-ready Arabic RTL templates for Tager supplier invoices and purchase orders.

Open `index.html`, or deploy the folder to GitHub Pages.

Main files:

- `invoice.html`
- `purchase-order.html`
- `assets/css/tager-documents.css`
- `assets/js/tager-documents.js`
- `data/sample-data.json`
- `assets/reference/` original reference PNG files
- `exact/` original image pages

Arabic integration instructions are in `README_AR.md`.
