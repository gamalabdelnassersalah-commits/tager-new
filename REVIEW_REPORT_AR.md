# تقرير المراجعة

تمت مراجعة الصفحات الأساسية: الرئيسية، السوق، الموردين، السلة، حساب العميل، لوحة المورد، لوحة الإدارة، الدعم، والسياسات.

تمت مراجعة منطق الطلبات والتوصيل والحسابات:
- لا يتم الطلب خارج تغطية المورد.
- لا يتم احتساب عمولة المنصة كمستحقة إلا بعد التسليم.
- لا تخصم دفعة المورد إلا بعد اعتماد الإدارة.
- يتم خصم المخزون عند الطلب وإرجاعه عند الإلغاء.
