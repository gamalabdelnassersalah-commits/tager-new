# سجل تغييرات V24

## إصلاحات التصميم
- تم إنشاء نسخة مقصوصة ومناسبة من اللوجو الكامل.
- تم إنشاء نسخة أيقونة منفصلة لاستخدامها في Favicon وManifest.
- تم إصلاح عرض اللوجو في Header وFooter.
- تم تحسين حالات الصفحات الفارغة بتصميم احترافي.

## إصلاحات المنطق الإداري
- تم نقل عمولة المنصة ورسوم السلة المميزة إلى صفحة إعدادات الإدارة فقط.
- تم حذف هذه الحقول من نموذج تسجيل المورد.
- تم إضافة رسالة واضحة في نموذج المورد أن المراجعة المالية شأن إداري داخلي.

## استكمال الصفحات
- صفحة رئيسية أوسع تعرض مسارات المنصة كاملة.
- صفحة منتجات بها شرح مسار الشراء حتى عند عدم وجود منتجات.
- صفحة موردين بها بحث وفلترة وحالة فارغة واضحة.
- صفحة سياسات أعيدت صياغتها بدون كشف إعدادات إدارية.

## فحص
- `node --check app.js` ناجح.
- `node --check supabase-client.js` ناجح.
- `npm run build` ناجح.
