# سجل تطوير V26

## أهم التطويرات
- إعادة تصميم الواجهة بالكامل بدرجة أعلى من الفخامة والوضوح.
- إضافة صفحة الفئات.
- إضافة صفحة تفاصيل المنتج.
- تحسين الصفحة الرئيسية ومركز التشغيل.
- تحسين لوحة الإدارة وإضافة مشرفين.
- توسيع إعدادات المنصة: حالة المنصة، SLA التشغيل، طرق الدفع، أرقام الدعم.
- تحسين حساب رسوم التوصيل حسب مناطق تغطية المورد.
- تحسين صفحات الحالة الفارغة حتى لا تظهر المنصة ناقصة.
- إضافة قوالب CSV فارغة.
- تحديث نسخة التخزين إلى V26 حتى تبدأ النسخة نظيفة.
