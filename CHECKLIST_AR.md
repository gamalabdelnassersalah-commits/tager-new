# قائمة فحص التشغيل

- إنشاء حساب الإدارة.
- تثبيت بيانات الاتصال والواتساب.
- مراجعة العملة والدولة واسم العملة الظاهر.
- مراجعة العمولات ورسوم السلة والضريبة داخل الإعدادات المالية.
- مراجعة بادئة الطلب وبادئة الفاتورة.
- إضافة الفئات المطلوبة.
- تسجيل الموردين واعتمادهم.
- إضافة مناطق تغطية لكل مورد مع رسوم التوصيل وأيام التسليم.
- إضافة المنتجات وربطها بالمورد والفئة والسعر والمخزون.
- نشر المنتجات بعد المراجعة.
- اختبار طلب كامل من السلة إلى الإرسال.
- اختبار تتبع الطلب برقم الطلب ورقم الهاتف.
- مراجعة مركز التشغيل والتنبيهات.
- مراجعة مركز الاعتمادات.
- مراجعة المخزون وحد التنبيه.
- إصدار فاتورة لطلب مسلم.
- تسجيل مرتجع اختباري إن لزم.
- مراجعة مركز المالية والتسويات.
- اختبار تصدير CSV وملف JSON.
- مراجعة صفحة السياسات والدعم.
- رفع مجلد public على الاستضافة.
