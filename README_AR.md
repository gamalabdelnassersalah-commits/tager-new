# Tager Final Release V18

نسخة تشغيل نهائية مطورة للمنصة بتصميم راقٍ وربط Supabase وVercel.

## المحتوى
- واجهة عربية RTL احترافية.
- صفحات رئيسية مباشرة: الرئيسية، السوق، الموردون، السلة، العميل، المورد، الإدارة، الدعم، السياسات.
- مركز متابعة تشغيل داخل لوحة الإدارة.
- مالية الموردين والعملاء والتوصيلات.
- احتساب عمولة المنصة بعد التسليم فقط.
- متابعة المدفوع، تحت المراجعة، المتبقي، والمنفذ.
- منع الطلب إذا كان المورد لا يغطي عنوان العميل.
- تصدير CSV للتقارير الأساسية.

## الرفع
ارفع محتويات المجلد فقط إلى GitHub، ثم شغّل `supabase/schema_final.sql` في Supabase، وانتظر Vercel حتى يصبح Ready.
