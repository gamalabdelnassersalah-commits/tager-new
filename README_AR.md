# Tager V26 Elite Production Platform

نسخة إنتاجية مطورة بتصميم عربي RTL راقٍ، تبدأ بدون موردين أو منتجات أو طلبات جاهزة.

## التشغيل السريع
1. فك ضغط الملف.
2. شغّل: `npm run check`
3. شغّل: `npm run build`
4. افتح مجلد `public` أو ارفعه على Vercel.
5. أول دخول يكون من `/setup` لإنشاء حساب الإدارة الحقيقي.

## ملاحظات مهمة
- لا توجد بيانات تجريبية داخل النسخة.
- إعدادات العمولة ورسوم السلة المميزة داخل لوحة الإدارة فقط.
- المورد والمنتج لا يظهران للعامة إلا بعد الاعتماد.
- الطلب لا يتم إذا كانت منطقة العميل خارج تغطية المورد.
- البيانات تحفظ محليًا داخل المتصفح في هذه النسخة، مع إمكانية تصدير واستيراد JSON.
