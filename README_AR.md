# Tager V25 Working Enterprise Platform

نسخة مطورة على نفس ملفات V10 ومجهزة للتشغيل المباشر على Vercel بدون شاشة بيضاء وبدون بيانات جاهزة أو حسابات مسبقة.

## التشغيل

1. ارفع الملفات على GitHub أو Vercel.
2. أمر البناء:
   `npm run build`
3. Output Directory:
   `public`
4. أول صفحة بعد التشغيل:
   `/setup`
5. أنشئ حساب الإدارة الحقيقي، ثم ابدأ اعتماد الموردين والمنتجات.

## الصفحات المكتملة

- الرئيسية
- المنتجات
- الموردون
- صفحة مورد عامة
- السلة
- إتمام الطلب
- تسجيل عميل
- تسجيل مورد
- تسجيل الدخول
- حساب العميل
- لوحة المورد
- لوحة الإدارة
- الدعم
- كيف تعمل المنصة
- السياسات

## الإدارة

تشمل لوحة الإدارة:

- اعتماد / رفض / إيقاف المستخدمين
- اعتماد الموردين
- اعتماد المنتجات
- متابعة الطلبات وتحديث الحالة
- الإدارة المالية ومتابعة مستحقات المنصة
- إعدادات المنصة المالية والتشغيلية
- تذاكر الدعم
- تصدير CSV و JSON

## ملاحظات مهمة

- لا توجد بيانات جاهزة داخل النسخة.
- لا توجد حسابات دخول مسبقة.
- إعدادات النسب المالية داخل لوحة الإدارة فقط.
- صفحات العملاء والموردين لا تعرض إعدادات مالية إدارية.
- اللوجو تم تثبيته داخل الهيدر والفوتر وأيقونة الموقع.
