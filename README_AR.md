# Tager V24 Enterprise Full Audit

نسخة إنتاجية شاملة بدون أي بيانات تجريبية داخل التشغيل.

## طريقة التشغيل
1. ارفع الملفات على Vercel أو أي استضافة Static.
2. افتح `/setup` لإنشاء حساب الإدارة الحقيقي لأول مرة.
3. سجل الموردين أو استورد البيانات من مركز البيانات.
4. اعتمد الموردين والمنتجات من لوحة الإدارة.
5. تابع التشغيل والمالية من صفحات الإدارة.

## ما تم ضبطه في V24
- إصلاح اللوجو الرسمي في الهيدر والفوتر والأيقونات والفاتورة.
- إزالة حقول عمولة المنصة ورسوم السلة المميزة من تسجيل المورد لأنها إعدادات إدارية داخلية.
- وضع الإعدادات المالية داخل صفحة إعدادات الإدارة فقط.
- استكمال صفحات المنتجات والموردين بحالات فارغة احترافية بدل الصفحات الفارغة.
- إضافة شرح تشغيل كامل داخل الصفحة الرئيسية.
- فصل الصفحات العامة عن صفحات الإدارة.
- مراجعة التنقل والظهور حسب صلاحية المستخدم.
- تحسين صفحة الموردين بالبحث والفلترة حسب المحافظة والمركز.
- تحسين صفحة المنتجات وشرح مسار الطلب والتوصيل.
- تحديث التقارير والملفات العامة إلى V24.

## الصفحات الرئيسية
- `/` الصفحة الرئيسية
- `/market` المنتجات
- `/vendors` الموردون
- `/cart` السلة
- `/support` الدعم
- `/policies` السياسات
- `/login` تسجيل الدخول
- `/setup` إنشاء أول حساب إدارة

## صفحات الإدارة بعد الدخول كمدير
- `/admin` لوحة الإدارة
- `/operations` مركز التشغيل
- `/finance` المركز المالي
- `/data-center` مركز البيانات والاستيراد والتصدير
- `/settings` إعدادات المنصة والإعدادات المالية الداخلية
- `/launch-check` جاهزية الإطلاق

## قواعد مهمة
- لا يوجد حسابات دخول جاهزة.
- لا يوجد موردون أو منتجات تجريبية.
- لا يظهر المورد إلا بعد اعتماد الإدارة.
- لا يظهر المنتج إلا بعد اعتماد الإدارة.
- الإعدادات المالية الداخلية لا تظهر في تسجيل المورد.
