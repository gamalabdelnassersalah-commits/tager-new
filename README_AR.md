# Tager — منصة تجارة وتوريد متكاملة

منصة عربية RTL لإدارة سوق توريد بين الموردين والعملاء، مع لوحة إدارة كاملة للتشغيل والمالية والاعتماد والدعم.

## التشغيل السريع
1. افتح المشروع.
2. شغّل الأمر: `npm install` ثم `npm run build`.
3. ارفع مجلد `public` على Vercel أو أي استضافة Static.
4. افتح `/setup` وأنشئ أول حساب إدارة حقيقي.

## لوحات التشغيل
- لوحة الإدارة: المستخدمون، الموردون، المنتجات، الفئات والمواد، الطلبات، المالية، التقارير، الإعدادات، الدعم، البيانات.
- لوحة المورد: بيانات المورد، مناطق التغطية، المنتجات، الطلبات، الحساب المالي، الدفعات.
- لوحة العميل: البيانات، العناوين، الطلبات، المتابعة.

## ملاحظات مهمة
- لا توجد حسابات دخول جاهزة.
- لا توجد منتجات أو موردون جاهزون.
- عمولة المنصة ورسوم السلة المميزة داخل لوحة الإدارة فقط.
- المورد لا يظهر للعامة إلا بعد اعتماد الإدارة.
- المنتج لا يظهر في السوق إلا بعد اعتماد الإدارة.
- الطلب لا يتم إذا كانت منطقة العميل خارج تغطية المورد.
