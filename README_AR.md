# Tager - إصدار تشغيل متكامل

هذه النسخة مخصصة للرفع على Vercel وربط Supabase.

## المحتوى
- واجهة عربية احترافية.
- إدارة الموردين والمنتجات والطلبات والتوصيل.
- تحقق من مناطق التوصيل قبل إتمام الطلب.
- حسابات مالية للمورد والعميل والإدارة.
- احتساب عمولة المنصة بعد التسليم فقط.
- خصم المخزون عند إنشاء الطلب وإرجاعه عند الإلغاء مرة واحدة.
- طباعة فاتورة الطلب.
- إدارة رسائل الدعم.
- إضافة فريق إدارة بصلاحيات تشغيلية.
- تصدير ملفات CSV.

## التشغيل
ارفع محتويات المجلد إلى GitHub، ثم شغل ملف قاعدة البيانات `supabase/schema_final.sql` في Supabase.
