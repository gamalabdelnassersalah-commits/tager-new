# منصة Tager

منصة تجارة جملة بواجهة عربية، تصميم فاتح، ولوحات منفصلة للعميل والمورد والإدارة.

## التشغيل المحلي
افتح الملف `index.html` مباشرة، أو شغل:

```bash
npm run build
```

ثم ارفع مجلد `public`.

## إعدادات Vercel
Build Command:

```bash
npm run build
```

Output Directory:

```bash
public
```

## ترتيب الحسابات
- الزائر يرى الصفحات العامة فقط: الرئيسية، المنتجات، الفئات، الموردون، التغطية، تتبع الطلب، الدعم.
- العميل بعد الدخول يرى صفحات الشراء والسلة والتتبع والدعم.
- المورد بعد الدخول يرى المنتجات والأسعار والتغطية والطلبات والحساب المالي.
- الإدارة بعد الدخول ترى الاعتمادات والطلبات والمالية والإعدادات والتقارير.
