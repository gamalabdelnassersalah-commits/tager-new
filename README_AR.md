# منصة Tager

منصة سوق جملة بواجهة عربية بسيطة ومنظمة.

## ما تم ترتيبه
- الصفحة الرئيسية للزائر فقط: منتجات، موردون، فئات، تغطية، سلة، تتبع، دعم.
- تسجيل العميل منفصل عن تسجيل المورد.
- بعد الدخول تظهر الصفحات حسب نوع الحساب فقط.
- المنتجات تدعم صورة وسعر قطاعي وسعر جملة وسعر جملة الجملة.
- أدوات الإدارة والإعدادات المالية داخل لوحة الإدارة فقط.

## النشر على Vercel
Build Command: `npm run build`

Output Directory: `public`

## فواتير المورد
- داخل حساب المورد تظهر صفحة **فواتيري**.
- المورد يستطيع عرض وطباعة الفاتورة الخاصة بأصنافه فقط داخل الطلب.
- الفاتورة تعرض رقم الطلب، بيانات المورد، بيانات العميل، البنود، نوع السعر، الكمية، سعر الوحدة، الإجمالي، وصافي مستحق المورد التقديري.
