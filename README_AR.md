منصة Tager جاهزة للتشغيل والنشر، بنفس ملفات المشروع ونفس التصميم، مع إضافة نظام لايف لمتابعة الطلبات والاعتمادات والدعم والجاهزية.

التشغيل المحلي: افتح index.html أو استخدم npm run build ثم ارفع مجلد public.


## تحديث الأسعار والصور
- تم الحفاظ على نفس أسماء الملفات ونفس اتجاه التصميم والألوان.
- أصبح لكل منتج ثلاثة أسعار: سعر القطاعي، سعر الجملة، وسعر جملة الجملة.
- تم إضافة صورة المنتج من الجهاز أو من رابط صورة.
- تم تبسيط الصفحة الرئيسية لتكون أوضح وأسهل للعميل والمورد.
- لا تظهر المنتجات إلا بعد اعتماد الإدارة.
