# منصة تاجر

ملف تشغيل منصة تاجر بصفحة واحدة مع الحفاظ على أسماء ملفات المشروع والتصميم الفاتح وألوان الهوية.

## التشغيل المحلي
افتح ملف `index.html` مباشرة أو شغل البناء:

```bash
npm run build
```

ثم ارفع مجلد `public` أو استخدم إعدادات Vercel التالية:

- Build Command: `npm run build`
- Output Directory: `public`

## ما تم تغطيته
- الصفحة الرئيسية
- المنتجات
- الفئات
- الموردون
- مناطق التغطية
- السلة
- تتبع الطلب
- الدعم
- السياسات
- لوحة العميل
- لوحة المورد
- لوحة الإدارة
- الإعدادات المالية داخل الإدارة فقط
