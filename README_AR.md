# منصة Tager

منصة تجارة جملة بواجهة عربية RTL وتصميم فاتح بنفس ألوان اللوجو.

## ما تم تضمينه
- الصفحة الرئيسية مبسطة وواضحة.
- صفحات عامة مهمة: المنتجات، الفئات، الموردون، التغطية، السلة، تتبع الطلب، الدعم، السياسات.
- صفحات العميل بعد الدخول: الشراء، السلة، الطلبات، التتبع، الدعم، الموردون، التغطية.
- صفحات المورد بعد الدخول: المنتجات، صور المنتجات، سعر القطاعي، سعر الجملة، سعر جملة الجملة، التغطية، الطلبات، الحساب المالي.
- صفحات الإدارة: المستخدمون، الموردون، المنتجات، الطلبات، المالية، الإعدادات، الدعم، البيانات، النظام لايف.
- الإعدادات المالية داخل الإدارة فقط.
- بدون بيانات جاهزة مسبقاً.
- جاهز للنشر على Vercel.

## التشغيل
افتح `index.html` مباشرة أو ارفع المشروع على Vercel.

إعدادات Vercel:
- Build Command: `npm run build`
- Output Directory: `public`
