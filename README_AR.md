# منصة Tager V23 Enterprise Production

نسخة إنتاجية شاملة بدون أي بيانات تجريبية أو حسابات جاهزة. تبدأ المنصة فارغة، وأول خطوة هي إنشاء حساب الإدارة من صفحة `/setup`.

## ما تم تطويره في V23

- مركز تشغيل يومي لمتابعة الطلبات المفتوحة، التوصيلات، المخزون المنخفض، الموردين والمنتجات المعلقة، والدعم.
- مركز مالية متقدم لمتابعة إجمالي الطلبات، المنفذ، المسدد، الملغي، عمولات المنصة، المتبقي على الموردين، وأعمار المديونية.
- مركز بيانات لاستيراد المنتجات ومناطق التوصيل عبر CSV وتصدير المنتجات، الموردين، الطلبات، والدفعات.
- صفحة إعدادات المنصة للتحكم في العمولات الافتراضية، أرقام الدعم، الواتساب، طرق الدفع، التصنيفات، وبيانات الحساب البنكي.
- قائمة جاهزية الإطلاق لتوضيح خطوات التشغيل قبل النشر الحقيقي.
- تحديث قاعدة البيانات بإضافات إنتاجية للمواد، التصنيفات، مستندات الموردين، وحقول مالية إضافية للمنتجات.
- استمرار تشغيل المنصة بدون بيانات تجريبية، مع إمكانية العمل المحلي الفارغ أو ربط Supabase.

## التشغيل السريع

1. ارفع الملفات على Vercel أو أي استضافة Static.
2. افتح `/setup` وأنشئ حساب الإدارة الحقيقي.
3. سجل الموردين الحقيقيين أو أضفهم من واجهات المنصة.
4. اعتمد الموردين من لوحة الإدارة.
5. أضف المنتجات ومناطق التوصيل.
6. راجع مركز التشغيل ومركز المالية يوميًا.

## ملفات مهمة

- `schema_final.sql` ملف قاعدة البيانات النهائي.
- `supabase/schema_final.sql` نسخة Supabase.
- `data/` قوالب CSV فارغة.
- `docs/` تقارير التسليم والتشغيل.

## ملاحظة مهمة

النسخة لا تحتوي على حسابات دخول جاهزة ولا منتجات جاهزة ولا موردين جاهزين. أي بيانات تظهر داخل المنصة تكون من إدخال الإدارة أو المستخدمين فقط.
