# منصة Tager

منصة تجارة جملة عربية RTL تعمل من صفحة واحدة، مع الحفاظ على نفس التصميم الفاتح وألوان Tager.

## التشغيل

افتح `index.html` مباشرة، أو ارفع المشروع على Vercel بالإعدادات التالية:

- Build Command: `npm run build`
- Output Directory: `public`
- Framework: Other

## الصفحات الأساسية

الرئيسية، المنتجات، الفئات، الموردون، التغطية، تتبع الطلب، السلة، الدعم، السياسات، تسجيل عميل، تسجيل مورد، تسجيل دخول، إعداد حساب الإدارة.

## بعد تسجيل الدخول

- العميل: شراء، سلة، مفضلة، طلب عرض سعر، تتبع، رسائل، مرتجع / شكوى، دعم.
- المورد: منتجات، صور، أسعار قطاعي / جملة / جملة الجملة، مناطق تغطية، مخزون، مستندات، رسائل، عروض أسعار، فواتير، حساب مالي.
- الإدارة: اعتماد موردين ومنتجات، طلبات، مالية، إعدادات، تسويات، عروض وكوبونات، شحن، فواتير، رقابة، مستندات، رسائل، النظام لايف.

## ملاحظات مهمة

- لا توجد بيانات تجريبية جاهزة.
- أول خطوة هي إنشاء حساب الإدارة من صفحة الإعداد.
- الإعدادات المالية داخل الإدارة فقط.
- اللوجو محفوظ داخل الملفات ويظهر في الواجهة والفواتير.
