# ابدأ من هنا

افتح الرابط بعد النشر ثم اتبع المسار التالي:

1. `/setup` إنشاء حساب الإدارة.
2. `/settings` ضبط أرقام الدعم، الواتساب، التصنيفات، وطرق الدفع.
3. `/admin` اعتماد الموردين والمنتجات.
4. `/data-center` استيراد المنتجات أو مناطق التوصيل عند الحاجة.
5. `/operations` متابعة التشغيل اليومي.
6. `/finance` متابعة المالية والعمولات والتحصيل.
7. `/launch-check` مراجعة جاهزية الإطلاق.

لا توجد بيانات تجريبية داخل النسخة. المنصة تبدأ فارغة بالكامل.
