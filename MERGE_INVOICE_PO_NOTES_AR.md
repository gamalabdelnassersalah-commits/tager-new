# دمج تصميمات الفاتورة وطلب الشراء

تم دمج ملفات التصميم داخل مجلد `tager-documents` بدون حذف أو تغيير بيانات المنصة الأساسية.

## ما تم
- إضافة تصميم الفاتورة وطلب الشراء كملفات مستقلة.
- ربط صفحات الطباعة داخل المنصة بإطار عرض آمن iframe.
- عدم تغيير بيانات العملاء أو الموردين أو الطلبات أو الفواتير.
- الحفاظ على وظائف المنصة الأساسية.

- `tager-documents/invoice.html`
- `tager-documents/purchase-order.html`
- `tager-documents/assets/css/tager-documents.css`
- `tager-documents/assets/js/tager-documents.js`
